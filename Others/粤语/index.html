# 粤语学习

# 声调

粤语声调有6个，前3个为阳调，后3个为阴调

1 伊

2 儿

3 三

4 死

5 无

6 溜

# 代词

# 声母

声母可以叠加声调发音

发音类似于拼音，区别是在拼音后面加上o

ng 粤语我的发音

gw 普通话锅的发音

kw 普通话括的第一声发音
